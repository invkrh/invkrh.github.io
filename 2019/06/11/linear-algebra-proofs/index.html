Linear Algebra Proofs

Posted on 2019-06-11 | In Math | Comments: | Views:

A list of useful proofs on linear algebra

Note: matrix cookbook is an extremely useful material when you deal with derivatives of matrices and vectors.

1. Symetric Matrix

Definition

Matrix AAA is symmetric if A=ATA = A^TA=AT

Proof 1.1

If λ\lambdaλ is the eigen-value of AAA, so is λ‾\overline{\lambda}λ
If xxx is the eigen-vector of AAA, so is x‾\overline{x}x
(bar means conjugate)

Since AAA is a real matrix,

A=A‾A = \overline{A} A=A

Knowing that Ax=λxAx = \lambda xAx=λx

Ax‾=A‾x‾=Ax‾=λx‾=λ‾x‾A \overline{x} = \overline{A} \overline{x}=\overline{A x}=\overline{\lambda x}=\overline{\lambda} \overline{x} Ax=Ax=Ax=λx=λx

Proof 1.2

AAA has only real eigenvalues

Consider λ\lambdaλ and xxx are the eigne-value and eigen-vector of A, repectively. Based on Proof 1.1,

x‾TAx=x‾Tλx=λx‾Tx\overline{x}^{T} A x=\overline{x}^{T} \lambda x=\lambda \overline{x}^{T} x xTAx=xTλx=λxTx

Since AxAxAx is a vector and x‾TAx=(Ax)Tx‾\overline{x}^{T} A x = (Ax)^T\overline{x}xTAx=(Ax)Tx,

x‾TAx=xTATx‾=xTAx‾=λ‾xTx‾\overline{x}^{T} A x = x^T A^T \overline{x} = x^T A \overline{x} = \overline{\lambda} x^T \overline{x} xTAx=xTATx=xTAx=λxTx

Since x‾Tx=xTx‾\overline{x}^{T} x = x^T \overline{x}xTx=xTx,

λ=λ‾\lambda = \overline{\lambda} λ=λ

Proof 1.3

AAA is diagonalizable by an orthogonal matrix.

Schur decomposition:
Every square matrix factors into A=QTQ−1A=QTQ^{-1}A=QTQ−1 where TTT is upper triangular and Q‾T=Q−1\overline{Q}^T=Q^{-1}Q​T=Q−1. If AAA has real eigenvalues then QQQ and TTT can be chosen real: QTQ=IQ^TQ = IQTQ=I (a.k.a QQQ is an orthogonal matrix)

Based on Proof 1.2, all the eigen values of AAA are real.
Based on Schur decomposition, QTAQ=TQ^TAQ = TQTAQ=T.
Then,

TT=(QTAQ)T=(AQ)TQ=QTATQ=QTAQ=TT^T = (Q^TAQ)^T = (AQ)^TQ = Q^TA^TQ = Q^TAQ = T TT=(QTAQ)T=(AQ)TQ=QTATQ=QTAQ=T

Denote the diagonal matrix T as D, we have

A=QDQTA = QDQ^T A=QDQT

Proof 1.4

If A is nonsingular, A−1A^{-1}A−1 is symmetric

Since A is invertible,

A−1A=IA^{-1}A = I A−1A=I

Taking the transpose, we have

I=IT=(A−1A)T=AT(A−1)T=A(A−1)T\begin{aligned} I &=I^{T}=\left(A^{-1} A\right)^{T} \\ &=A^{T}\left(A^{-1}\right)^{T} \\ &=A\left(A^{-1}\right)^{T} \end{aligned} I​=IT=(A−1A)T=AT(A−1)T=A(A−1)T​

Hence, A−1=(A−1)TA^{-1} = (A^{-1})^TA−1=(A−1)T

2. Positive definite symmetric matrix

Definition

A real symmetric n×nn \times nn×n matrix AAA is called positive definite if xTAx>0x^TAx>0xTAx>0 for all non-zero vectors x∈Rnx \in \mathbb{R}^nx∈Rn.

Proof 2.1

The eigenvalues of a real symmetric positive-definite matrix A are all positive.

Let λ\lambdaλ be a (real) eigenvalue of AAA and let xxx be a corresponding real eigenvector. That is, we have

Ax=λxAx=\lambda x Ax=λx

Then we multiply by xTx^TxT on left and obtain,

xTAx=λxTx=λ∥x∥2\begin{aligned} x^T A x &=\lambda x^T x \\ &=\lambda\|x\|^{2} \end{aligned} xTAx​=λxTx=λ∥x∥2​

The left hand side is positive as AAA is positive definite and xxx is a nonzero vector as it is an eigenvector.
Since the norm ∥x∥2\|x\|^2∥x∥2 is positive, we must have λ\lambdaλ is positive.
It follows that every eigenvalue λ\lambdaλ of AAA is real.

Proof 2.2

If eigenvalues of a real symmetric matrix A are all positive, then A is positive-definite.

Since Proof 1.3, A = QDQ^T where Q^{-1} = Q^T, we have

xTAx=xTQDQTxx^TAx = x^TQDQ^Tx xTAx=xTQDQTx

where

D=[λ10000λ200⋮⋯⋱⋮00⋯λn]D=\left[\begin{array}{cccc}{\lambda_{1}} & {0} & {0} & {0} \\ {0} & {\lambda_{2}} & {0} & {0} \\ {\vdots} & {\cdots} & {\ddots} & {\vdots} \\ {0} & {0} & {\cdots} & {\lambda_{n}}\end{array}\right] D=⎣⎢⎢⎢⎡​λ1​0⋮0​0λ2​⋯0​00⋱⋯​00⋮λn​​⎦⎥⎥⎥⎤​

Putting y=QTxy = Q^Txy=QTx, we can rewrite the above equation as

xTAx=yTDyx^TAx = y^TDy xTAx=yTDy

Let

y=[y1y2⋮yn]y=\left[\begin{array}{c}{y_{1}} \\ {y_{2}} \\ {\vdots} \\ {y_{n}}\end{array}\right] y=⎣⎢⎢⎢⎡​y1​y2​⋮yn​​⎦⎥⎥⎥⎤​

Then we have

xTAx=yTDy=λ1y12+λ2y22+⋯+λnyn2{x^{T} A x=y^{T} D y} = \lambda_{1} y_{1}^{2}+\lambda_{2} y_{2}^{2}+\cdots+\lambda_{n} y_{n}^{2} xTAx=yTDy=λ1​y12​+λ2​y22​+⋯+λn​yn2​

By assumption eigenvalues λi\lambda_iλi​ are positive.
Also, since xxx is a nonzero vector and QQQ is invertible, y=QTxy=QTxy=QTx is not a zero vector.
Thus the sum expression above is positive, hence xTAxx^TAxxTAx is positive for any nonzero vector xxx.
Therefore, the matrix AAA is positive-definite.

Proof 2.3

A is invertible

Method 1

Since Proof 2.1, the matrix AAA does not have 0 as an eigenvalue

We can prove this by contradiction:
If Ax=0⋅xAx=0 \cdot xAx=0⋅x for some x≠0x \neq 0x​=0 then by definition of eigenvalues (non-invertible), xxx is an eigenvector with eigenvalue λ=0\lambda = 0λ=0

Method 2

We can prove this by using determinent

det⁡A=∏inλi>0\operatorname{det} A = \prod_i^n \lambda_i \gt 0 detA=i∏n​λi​>0

Proof 2.4

the inverse of A is positive-definite

Based on Proof 2.1 and Proof 2.2, we know the fact that a symmetric matrix is positive-definite if and only if its eigenvalues are all positive.

All eigenvalues of A−1A^{−1}A−1 are of the form 1/λ1 / \lambda1/λ, where λ\lambdaλ is an eigenvalue of AAA.
Since A is positive-definite, each eigenvalue λ\lambdaλ is positive, hence 1/λ1 / \lambda1/λ is positive.

So all eigenvalues of A−1A^{-1}A−1 are positive, and it yields that A−1A^{-1}A−1 is positive-definite.

3. Matrix derivative

Proof 3.1

Knowing that AAA is an invertible symmetric matrix, then ∂log⁡det⁡(A)∂A=A−1\dfrac{\partial \log \operatorname{det}(A)}{\partial A} = A^{-1}∂A∂logdet(A)​=A−1

Since the chain rule,

∂log⁡det⁡(A)∂A=1det⁡(A)∂det⁡(A)∂A\begin{aligned} \frac{\partial \log \operatorname{det}(A)}{\partial A} &= \frac{1}{\operatorname{det}(A)} \frac{\partial \operatorname{det}(A)}{\partial A} \end{aligned} ∂A∂logdet(A)​​=det(A)1​∂A∂det(A)​​

According to the cookbook: ∂det⁡(A)∂A=det⁡(A)(A−1)T\dfrac{\partial \operatorname{det}(A)}{\partial A}=\operatorname{det}(A)\left(A^{-1}\right)^{T}∂A∂det(A)​=det(A)(A−1)T,

∂log⁡det⁡(A)∂A=(A−1)T\frac{\partial \log \operatorname{det}(A)}{\partial A} = \left(A^{-1}\right)^{T} ∂A∂logdet(A)​=(A−1)T

Since the proof 1,

∂log⁡det⁡(A)∂A=A−1\frac{\partial \log \operatorname{det}(A)}{\partial A} = A^{-1} ∂A∂logdet(A)​=A−1

Proof 3.2

Knowing that AAA is an invertible symmetric matrix, then ∂xTA−1x∂A=xxTA−2\dfrac{\partial x^T A^{-1} x}{\partial A} = x x ^T A^{-2}∂A∂xTA−1x​=xxTA−2
