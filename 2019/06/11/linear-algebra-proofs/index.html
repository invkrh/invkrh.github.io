Linear Algebra Proofs

Posted on 2019-06-11 | In Math | Comments: | Views:

A list of useful properties and their proofs in linear algebra. Some of them are also very useful in machine learning. Complex proofs are not in the scope of the post, but the references will be given if interested.

0. Basics

Proof 0.1

Let A and B be two N×NN \times NN×N square matrix, then det(AB)=det(A)⋅det(B)det(AB) = det(A) \cdot det(B)det(AB)=det(A)⋅det(B)

Proof: https://proofwiki.org/wiki/Determinant_of_Matrix_Product

Proof 0.2

The determinant of a orthogonal matrix must be ±1\pm1±1

Because 1=det(I)=det(QTQ)=det(QT)det(Q)=(det(Q))2⇒det(Q)=±11=det(I)=det\left(Q^{T} Q\right)=det\left(Q^{T}\right) det(Q)=(det(Q))^{2} \Rightarrow det(Q)=\pm 11=det(I)=det(QTQ)=det(QT)det(Q)=(det(Q))2⇒det(Q)=±1

Proof 0.3

Let AAA be an N×NN \times NN×N matrix and let λ1,...,λn\lambda_1, ..., \lambda_nλ1​,...,λn​ be its eigenvalues, then

det(A)=∏i=1nλidet(A)=\prod_{i=1}^{n} \lambda_{i} det(A)=i=1∏n​λi​

tr(A)=∑i=1nλitr(A)=\sum_{i=1}^{n} \lambda_{i} tr(A)=i=1∑n​λi​

Proof: https://yutsumura.com/determinant-trace-and-eigenvalues-of-a-matrix/

1. Symetric Matrix

Definition

Matrix AAA is symmetric if A=ATA = A^\mathrm{T}A=AT

Proof 1.1

If λ\lambdaλ is the eigen-value of AAA, so is the conjugate of λ\lambdaλ, denoted as λ‾\overline{\lambda}λ
If x\mathbf{x}x is the eigen-vector of AAA, so is the conjugate of x\mathbf{x}x, denoted as x‾\overline{\mathbf{x}}x

Since AAA is a real matrix,

A=A‾A = \overline{A} A=A

Knowing that Ax=λxA\mathbf{x} = \lambda \mathbf{x}Ax=λx

Ax‾=A‾x‾=Ax‾=λx‾=λ‾x‾A \overline{\mathbf{x}} = \overline{A} \overline{\mathbf{x}}=\overline{A \mathbf{x}}=\overline{\lambda \mathbf{x}}=\overline{\lambda} \overline{\mathbf{x}} Ax=Ax=Ax=λx=λx

Proof 1.2

AAA has only real eigenvalues

Consider λ\lambdaλ and x\mathbf{x}x are the eigne-value and eigen-vector of AAA, repectively. Based on Proof 1.1,

x‾TAx=x‾Tλx=λx‾Tx\overline{\mathbf{x}}^{\mathrm{T}} A \mathbf{x}=\overline{\mathbf{x}}^{\mathrm{T}} \lambda \mathbf{x}=\lambda \overline{\mathbf{x}}^{\mathrm{T}} \mathbf{x} xTAx=xTλx=λxTx

Since AxA\mathbf{x}Ax is a vector and x‾TAx=(Ax)Tx‾\overline{\mathbf{x}}^{\mathrm{T}} A \mathbf{x} = (A\mathbf{x})^\mathrm{T}\overline{\mathbf{x}}xTAx=(Ax)Tx,

x‾TAx=xTATx‾=xTAx‾=λ‾xTx‾\overline{\mathbf{x}}^{\mathrm{T}} A \mathbf{x} = \mathbf{x}^\mathrm{T} A^\mathrm{T} \overline{\mathbf{x}} = \mathbf{x}^\mathrm{T} A \overline{\mathbf{x}} = \overline{\lambda} \mathbf{x}^\mathrm{T} \overline{\mathbf{x}} xTAx=xTATx=xTAx=λxTx

Since x‾Tx=xTx‾\overline{\mathbf{x}}^{\mathrm{T}} \mathbf{x} = \mathbf{x}^\mathrm{T} \overline{\mathbf{x}}xTx=xTx,

λ=λ‾\lambda = \overline{\lambda} λ=λ

Proof 1.3

AAA is diagonalizable by an orthogonal matrix.

Schur decomposition:
Every square matrix factors into A=QTQ−1A=QTQ^{-1}A=QTQ−1 where TTT is upper triangular and Q‾T=Q−1\overline{Q}^\mathrm{T}=Q^{-1}Q​T=Q−1. If AAA has real eigenvalues then QQQ and TTT can be chosen real: QTQ=IQ^\mathrm{T}Q = IQTQ=I (a.k.a QQQ is an orthogonal matrix)

Based on Proof 1.2, all the eigen values of AAA are real.
Based on Schur decomposition, QTAQ=TQ^\mathrm{T}AQ = TQTAQ=T.
Then,

TT=(QTAQ)T=(AQ)TQ=QTATQ=QTAQ=TT^\mathrm{T} = (Q^\mathrm{T}AQ)^\mathrm{T} = (AQ)^\mathrm{T}Q = Q^\mathrm{T}A^\mathrm{T}Q = Q^\mathrm{T}AQ = T TT=(QTAQ)T=(AQ)TQ=QTATQ=QTAQ=T

Denote the diagonal matrix TTT as DDD, we have

A=QDQTA = QDQ^\mathrm{T} A=QDQT

Proof 1.4

If AAA is nonsingular, A−1A^{-1}A−1 is symmetric

Since AAA is invertible,

A−1A=IA^{-1}A = I A−1A=I

Taking the transpose, we have

I=IT=(A−1A)T=AT(A−1)T=A(A−1)T\begin{aligned} I &=I^{\mathrm{T}}=\left(A^{-1} A\right)^{\mathrm{T}} \\ &=A^{\mathrm{T}}\left(A^{-1}\right)^{\mathrm{T}} \\ &=A\left(A^{-1}\right)^{\mathrm{T}} \end{aligned} I​=IT=(A−1A)T=AT(A−1)T=A(A−1)T​

Hence, A−1=(A−1)TA^{-1} = (A^{-1})^\mathrm{T}A−1=(A−1)T

2. Positive definite symmetric matrix

Definition

A real symmetric n×nn \times nn×n matrix AAA is called positive definite if xTAx>0\mathbf{x}^\mathrm{T}A\mathbf{x}>0xTAx>0 for all non-zero vectors x∈Rn\mathbf{x} \in \mathbb{R}^nx∈Rn.

Proof 2.1

The eigenvalues of a real symmetric positive-definite matrix AAA are all positive.

Let λ\lambdaλ be a (real) eigenvalue of AAA and let x\mathbf{x}x be a corresponding real eigenvector. That is, we have

Ax=λxA\mathbf{x}=\lambda \mathbf{x} Ax=λx

Then we multiply by xT\mathbf{x}^\mathrm{T}xT on left and obtain,

xTAx=λxTx=λ∥x∥2\begin{aligned} \mathbf{x}^\mathrm{T} A \mathbf{x} &=\lambda \mathbf{x}^\mathrm{T} \mathbf{x} \\ &=\lambda\|\mathbf{x}\|^{2} \end{aligned} xTAx​=λxTx=λ∥x∥2​

The left hand side is positive as AAA is positive definite and xxx is a nonzero vector as it is an eigenvector.
Since the norm ∥x∥2\|\mathbf{x}\|^2∥x∥2 is positive, we must have λ\lambdaλ is positive.
It follows that every eigenvalue λ\lambdaλ of AAA is real.

Proof 2.2

If eigenvalues of a real symmetric matrix AAA are all positive, then AAA is positive-definite.

Since Proof 1.3, A=QDQTA = QDQ^\mathrm{T}A=QDQT where Q−1=QTQ^{-1} = Q^\mathrm{T}Q−1=QT, we have

xTAx=xTQDQTx\mathbf{x}^\mathrm{T}A\mathbf{x} = \mathbf{x}^\mathrm{T}QDQ^\mathrm{T}\mathbf{x} xTAx=xTQDQTx

where

D=[λ10000λ200⋮⋯⋱⋮00⋯λn]D=\left[\begin{array}{cccc}{\lambda_{1}} & {0} & {0} & {0} \\ {0} & {\lambda_{2}} & {0} & {0} \\ {\vdots} & {\cdots} & {\ddots} & {\vdots} \\ {0} & {0} & {\cdots} & {\lambda_{n}}\end{array}\right] D=⎣⎢⎢⎢⎡​λ1​0⋮0​0λ2​⋯0​00⋱⋯​00⋮λn​​⎦⎥⎥⎥⎤​

Putting y=QTxy = Q^\mathrm{T}\mathbf{x}y=QTx, we can rewrite the above equation as

xTAx=yTDy\mathbf{x}^\mathrm{T}A\mathbf{x} = \mathbf{y}^\mathrm{T}D\mathbf{y} xTAx=yTDy

Let

y=[y1y2⋮yn]\mathbf{y}=\left[\begin{array}{c}{y_{1}} \\ {y_{2}} \\ {\vdots} \\ {y_{n}}\end{array}\right] y=⎣⎢⎢⎢⎡​y1​y2​⋮yn​​⎦⎥⎥⎥⎤​

Then we have

xTAx=yTDy=λ1y12+λ2y22+⋯+λnyn2\mathbf{x}^{\mathrm{T}} A x=\mathbf{y}^{\mathrm{T}} D \mathbf{y} = \lambda_{1} y_{1}^{2}+\lambda_{2} y_{2}^{2}+\cdots+\lambda_{n} y_{n}^{2} xTAx=yTDy=λ1​y12​+λ2​y22​+⋯+λn​yn2​

By assumption eigenvalues λi\lambda_iλi​ are positive.
Also, since xxx is a nonzero vector and QQQ is invertible, y=QTxy=QTxy=QTx is not a zero vector.
Thus the sum expression above is positive, hence xTAx\mathbf{x}^\mathrm{T}A\mathbf{x}xTAx is positive for any nonzero vector x\mathbf{x}x.
Therefore, the matrix AAA is positive-definite.

Proof 2.3

A is invertible

Method 1

Since Proof 2.1, the matrix AAA does not have 0 as an eigenvalue

We can prove this by contradiction:
If Ax=0⋅xA\mathbf{x}=0 \cdot \mathbf{x}Ax=0⋅x for some x≠0\mathbf{x} \neq 0x​=0 then by definition of eigenvalues (non-invertible), x\mathbf{x}x is an eigenvector with eigenvalue λ=0\lambda = 0λ=0

Method 2

We can prove this by using determinent

detA=∏inλi>0det A = \prod_i^n \lambda_i \gt 0 detA=i∏n​λi​>0

Proof 2.4

the inverse of A is positive-definite

Based on Proof 2.1 and Proof 2.2, we know the fact that a symmetric matrix is positive-definite if and only if its eigenvalues are all positive.

All eigenvalues of A−1A^{−1}A−1 are of the form 1/λ1 / \lambda1/λ, where λ\lambdaλ is an eigenvalue of AAA.
Since A is positive-definite, each eigenvalue λ\lambdaλ is positive, hence 1/λ1 / \lambda1/λ is positive.

So all eigenvalues of A−1A^{-1}A−1 are positive, and it yields that A−1A^{-1}A−1 is positive-definite.
